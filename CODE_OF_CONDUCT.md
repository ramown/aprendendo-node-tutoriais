# Código de Conduta

## Nosso compromisso

Queremos manter este projeto aberto, respeitoso e colaborativo para todos.

## Comportamentos esperados

- respeito mútuo
- comunicação clara e cordial
- abertura para feedback
- colaboração construtiva

## Comportamentos inaceitáveis

- assédio ou intimidação
- linguagem ofensiva
- ataques pessoais
- exposição indevida de dados privados

## Aplicação

Casos de comportamento inadequado podem ser reportados aos mantenedores do projeto.

## Observação

Este arquivo pode ser expandido futuramente conforme a comunidade crescer.
