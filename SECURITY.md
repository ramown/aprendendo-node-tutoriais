# Política de Segurança

## Como reportar vulnerabilidades

Caso você encontre uma vulnerabilidade de segurança, não abra uma issue pública.

Entre em contato com o mantenedor por um canal privado e forneça:

- descrição da vulnerabilidade
- impacto potencial
- passos para reproduzir
- sugestão de mitigação, se houver

## Escopo

Esta política se aplica ao conteúdo, scripts e configurações deste repositório.

## Boas práticas

- não exponha segredos em commits
- use arquivos `.env` para dados sensíveis
- revise dependências antes de instalar em projetos reais
