# Nível 2 - Manipulação de Diretórios

- Criar e remover diretórios

- Identificar tipos de arquivos
- Renomear e mover arquivos

- mkdir
- readdir

- rename
- unlink / rm

- Criar scripts de automação
- Manipular estruturas de pastas
