# Configuração do Projeto

## Requisitos

- Node.js 20+
- npm 10+ ou equivalente
- Git

## Instalação inicial

```bash
git clone {{REPO_URL}}
cd aprendendo-node-tutoriais
npm install
```

## Scripts disponíveis

- `npm start`: inicia o menu interativo placeholder
- `npm run dev`: executa o placeholder em modo de desenvolvimento
- `npm test`: placeholder para testes
- `npm run lint`: placeholder para lint
- `npm run new-tutorial`: placeholder para automação futura

## Editor recomendado

Qualquer editor com suporte a JavaScript, Markdown e JSON.

## Observações

Este repositório começa como estrutura base. Os conteúdos dos tutoriais serão adicionados progressivamente.
