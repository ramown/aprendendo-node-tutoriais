# Estrutura do Repositório

## Organização geral

O repositório está dividido em módulos progressivos.

- `00-comece-aqui/`: visão geral e orientação inicial
- `01-fundamentos-javascript/`: base da linguagem
- `02-nodejs-essencial/`: conceitos centrais do Node.js
- `03-sistema-de-arquivos/`: leitura, escrita e manipulação de diretórios
- `04-servidores-http/` a `09-deploy/`: trilha backend progressiva
- `desafios-semanais/`: prática recorrente
- `mini-projetos/`: consolidação prática
- `recursos/`: materiais de apoio
- `scripts/`: automações e utilitários
- `docs/`: materiais complementares

## Padrão dos níveis

Cada pasta `nivelX-*` contém:

- `README.md`
- `exercicios.md`
- `projeto.md`

## Objetivo

Manter uma estrutura previsível, simples de navegar e fácil de evoluir.
