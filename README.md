## Pré-requisitos

- Node.js 20+
- npm 10+ ou equivalente
- Git
- Editor de código de sua preferência

## Como começar

1. Clone o repositório
2. Instale as dependências quando necessário
3. Siga a trilha em ordem numérica
4. Resolva os exercícios e avance para os projetos

```bash
git clone {{REPO_URL}}
cd aprendendo-node-tutoriais
npm install
npm start
```
